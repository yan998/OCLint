# OCLint
OCLint  基础配置,当前配置的对应官方0.15版本

1、第一步
```
cd /usr/local/bin
```
将下载项目的  bin  目录中的东西，拷贝到  /usr/local/bin 中

2、第二步
```
cd /usr/local/lib
```
将下载项目的  lib  目录中的东西<clang 和 oclint>，拷贝到  /usr/local/lib 中

3、重启终端，输入 oclint --version
```
  OCLint (http://oclint.org/):
  OCLint version 0.15.
  Built Feb 21 2020 (12:56:28).
```
安装成功
